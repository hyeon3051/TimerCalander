# TimerCalander
======
## About
계획을 만들고 안 지키는 사람이 많다고 합니다. 자신의 의지에 달린 문제이기도 하지만 도구를 사용하면 이를 편하게 진행 할 수 있습니다.
구글에서는 타임 타이머라는 스톱워치를 사용하여 시간을 지나는 것을 확실히 확인하고 그 시간 내에 계획을 지킬 수 있게 도와 줍니다.
여기서 영감을 얻어 타임 타이머와 캘린더를 합쳐 캘린더를 기록하고 10분 전후로 시작 버튼을 누르고 타임 타이머를 표현하고 다 끝나면 이를 기록에 남깁니다. 
내가 계획을 지켰을때 업적도 달성하게 하고 더하여 그 날마다 지킨 비율을 나타낼 수 있게 합니다.
